# Headline style

Load before naming page clusters in Phase 1 and sidebar titles in Phases 2–4.

## Rule

Name by the **action the reader takes**, not the bare noun from code.

| Avoid | Prefer |
| --- | --- |
| Theme | Customize theme and branding |
| llms.txt | Export docs for LLMs |
| Configuration | Configure your docs site |

## Reference pages

Grouped reference titles can be domain nouns: "Agent & MCP APIs", "LLM exports". Component pages use the component name (`Steps`, `Accordion`) for lookup.

## Banned in titles

Value adjectives: *easy*, *powerful*, *seamless*, *comprehensive*. Show through action, don't tell.

## Comparisons

Neutral orientation: "vs Docusaurus" is fine. Avoid superiority claims in titles.
