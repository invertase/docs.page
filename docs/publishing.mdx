Publishing docs.page documentation requires only a public GitHub repository.

## Requirements

- `docs.json` at the repository root
- Documentation files under `docs/`
- Repository must be **public** (private repos are not supported on the free tier)

## After publishing

Your site is immediately available at:

```
https://docs.page/{owner}/{repository}
```

Optionally configure a [custom domain](/custom-domains) or preview [branches and PRs](/live-previews).
