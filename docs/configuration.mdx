Projects configure name, description, logo, favicon, theme colors, header, sidebar, tabs, anchors, social links, SEO, search, scripts, content rendering, and mustache variables via a docs.json file validated by Zod.

Create a `docs.json` file at your repository root. The schema is validated with Zod and exposed at `GET /schema.json`.

## Top-level keys

| Key | Purpose |
|-----|---------|
| `name` | Site title |
| `description` | Site description |
| `logo` | Header logo (light/dark) |
| `favicon` | Browser favicon |
| `theme` | Colors and default light/dark mode — see [Theme](/appearance/theme) |
| `tabs` | Top navigation tabs — see [Navigation](/navigation/sidebar) |
| `sidebar` | Page navigation — see [Navigation](/navigation/sidebar) |
| `search` | Algolia DocSearch — see [Search](/integrations/search) |
| `scripts` | Analytics scripts — see [Analytics](/integrations/analytics) |
| `content` | Content rendering options |
| `variables` | Mustache variables — see [Variables](/variables) |
