Renders nested sidebar groups and pages from docs.json, supporting grouped pages, icons, tab associations, locale-specific sidebars, and href links at the group level.

## Sidebar

The `sidebar` field accepts an array of groups. Each group has a `pages` array:

```json
{
  "sidebar": [
    {
      "group": "Getting Started",
      "tab": "root",
      "pages": [
        { "title": "Overview", "href": "/", "icon": "house" }
      ]
    }
  ]
}
```

Groups can nest. Assign `tab` to show a group only on a specific tab.

## Tabs

Top-level tabs split documentation into sections:

```json
{
  "tabs": [
    { "id": "root", "title": "Documentation", "href": "/" },
    { "id": "cli", "title": "CLI", "href": "/cli" }
  ]
}
```

The active tab is inferred from the longest matching `href` prefix.

See [Locales](/navigation/locales) for translated sidebars.
