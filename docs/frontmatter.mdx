YAML frontmatter at the top of each MDX file controls page metadata.

## Supported fields

| Field | Purpose |
|-------|---------|
| `title` | Page title (overrides config default) |
| `description` | Meta description |
| `image` | Social preview image for this page |
| `redirect` | Redirect to another URL â€” see [Redirects](/redirects) |
| `previous` | Override previous page link |
| `next` | Override next page link |
| `noindex` | Exclude from search engine indexing |

## Example

```mdx
---
title: My Page
description: A short summary
---

Page content here.
```
