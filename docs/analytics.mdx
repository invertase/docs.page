Tracks documentation pageviews to Plausible with owner and repository props in production, configurable per-project via scripts.plausible in docs.json.

## Per-project Plausible

```json
{
  "scripts": {
    "plausible": "your-domain.com"
  }
}
```

## Google Analytics / Tag Manager

```json
{
  "scripts": {
    "googleAnalytics": "G-XXXXXXXX",
    "googleTagManager": "GTM-XXXXXXX"
  }
}
```
## Implementation

- `website/src/plausible.ts` `trackPageRequest` — server-side tracking
- `website/src/middleware.ts` — invocation
- `api/src/config/models/scripts.ts` — per-project config
