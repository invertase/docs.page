Reference images and other static files with paths relative to your GitHub repository.

## Relative paths

```mdx
<Image src="/assets/screenshot.png" alt="Screenshot" />
```

## External URLs

Absolute URLs are used as-is:

```mdx
<Image src="https://example.com/logo.png" alt="Logo" />
```

Logo and favicon paths in `docs.json` follow the same rules. See [Configuration](/configuration).
