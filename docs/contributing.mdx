docs.page is a monorepo with three main packages:

| Package | Path | Purpose |
|---------|------|---------|
| API | `api/` | Bundling, webhooks, config parsing |
| Website | `website/` | Next.js documentation renderer |
| CLI | `packages/cli/` | `init` and `check` commands |

## Local development

```bash
npm run dev
```

Runs the API (`:8080`) and website (`:3000`) concurrently.

## MDX plugins

Remark and rehype plugins live in `api/src/bundler/plugins/`. See [Writing content](/writing-content).
