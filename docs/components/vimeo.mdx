Use the `<Vimeo />` component to embed Vimeo videos in your documentation.

## Example

<Vimeo id="253989945" />

```jsx
<Vimeo id="253989945" />
```

## Props

<Property name="id" type="string" required>
  The ID of the Vimeo video to display.
</Property>
