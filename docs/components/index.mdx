Add these components to any `.mdx` file under `docs/`. Plain Markdown handles paragraphs, lists, and links; components add structure for tabs, callouts, code examples, and API reference fields.

## Structure

<CardGroup cols={1}>
  <Card title="Tabs" icon="folder" href="/components/tabs">
    Organize related content in switchable panels — for example, per-platform instructions.
  </Card>
  <Card title="Accordion" icon="chevron-down" href="/components/accordion">
    Collapse long sections so readers expand only what they need.
  </Card>
  <Card title="Steps" icon="list-ol" href="/components/steps">
    Show numbered or icon-backed steps in tutorials and setup guides.
  </Card>
  <Card title="Card" icon="square" href="/components/card">
    Present linked tiles on overview pages and landing sections.
  </Card>
  <Card title="Heading" icon="heading" href="/components/heading">
    Add anchor links to headings for in-page navigation.
  </Card>
</CardGroup>
