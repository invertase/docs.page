> Browse the full docs.page component library for layout, emphasis, API documentation, and navigation elements available in your MDX pages.

docs.page supports [MDX](https://mdxjs.com/), so you can use JSX components alongside standard Markdown in any `.mdx` file. Components structure long pages, highlight important details, and document APIs more clearly than plain prose alone.

Use Markdown for simple paragraphs, lists, and links. Reach for components when you need tabbed examples, expandable sections, styled callouts, or structured reference fields.
