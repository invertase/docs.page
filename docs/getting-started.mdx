Get a docs.page site running in a few minutes.

## Prerequisites

- A GitHub account
- A public repository (or plan to publish publicly)
- Node.js 18+ if using the CLI locally

## Steps

<Steps>

<Step title="Scaffold files">

```bash
npx @docs.page/cli init
```

Creates `docs.json` and starter pages under `docs/`.

</Step>

<Step title="Preview locally">

Open [docs.page/preview](https://docs.page/preview) and select your project directory.

See [Local previewing](/local-previewing).

</Step>

<Step title="Publish">

Push `docs.json` and `docs/` to GitHub. Your docs are live at `docs.page/{owner}/{repo}`.

See [Publishing](/publishing).

</Step>

</Steps>
