The `@docs.page/cli` package scaffolds and validates documentation projects.

## Install

```bash
npm install -g @docs.page/cli
```

Or run without installing:

```bash
npx @docs.page/cli --help
```

## Commands

- [`init`](/cli/init) — scaffold `docs.json` and starter pages
- [`check`](/cli/check) — validate configuration, routes, and links
