# Release notes ŌĆö v2.0.0

**Release date:** 29-06-2026 (platform on `main` via [#488](https://github.com/invertase/docs.page/pull/488); `@docs.page/cli` [2.0.0](https://github.com/invertase/docs.page/releases/tag/cli-v2.0.0) on npm 2026-06-23)

docs.page **2.0.0** replaces the 1.0.0 stack with a single Next.js application, shared markdown bundler, and agent-facing surfaces (MCP, `llms.txt`, Ask AI).

**See also:** [CHANGELOG](../../CHANGELOG.md#200---2026-06-29) ┬Ę [v1.0.0 release notes](./v1.0.0.mdx)
