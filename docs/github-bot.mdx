On pull_request.opened webhook events, posts a comment on the PR with a link to the documentation preview URL, using a custom domain when configured in domains.json.

## Install

1. Install the [docs-page GitHub App](https://github.com/apps/docs-page)
2. Grant access to your documentation repository
3. Open a pull request — a comment with the preview URL is posted automatically

<Warning>

GitHub check runs for documentation builds are implemented but not yet enabled in production.

</Warning>
<Warning>

- pull_request.synchronize handler and GitHub check runs are implemented but commented out in the webhook router.

</Warning>
## Implementation

- `api/src/events/pull_request.opened.ts` — handler
- `api/src/routes/webhooks.github.ts` — event dispatch
