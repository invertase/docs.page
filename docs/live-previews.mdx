Documentation URLs support viewing content from a specific branch, commit hash, or pull request number using a tilde suffix on the repository segment (e.g. owner/repo~123).

## URL patterns

| Ref type | Example |
|----------|---------|
| Branch | `docs.page/org/repo~feature-branch/page` |
| Commit | `docs.page/org/repo~abc123.../page` |
| Pull request | `docs.page/org/repo~42/page` |

Install the [GitHub bot](/github-bot) to get PR preview links automatically.
## Surface

- **ui**: `docs.page/{owner}/{repo}~{ref}/{path}`
## Implementation

- `api/src/bundler/index.ts` `Bundler.getSource` — ref resolution
- `website/src/pages/[[...path]].tsx` — URL parsing
