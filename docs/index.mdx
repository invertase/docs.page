docs.page serves documentation directly from a public GitHub repository with zero hosting setup.

## URL structure

Published docs are available at:

```
https://docs.page/{owner}/{repository}/{path}
```

Preview a specific branch, commit, or pull request with a tilde suffix:

```
https://docs.page/{owner}/{repository}~{ref}/{path}
```

## What you need

1. A public GitHub repository
2. A `docs.json` file at the repository root
3. MDX or Markdown files under `docs/`

See [Getting Started](/getting-started) to create your first project.
