Compiles MDX content with remark and rehype plugins for GFM, component validation, undeclared variable detection, heading extraction, code blocks, slugs, inline badges, and accessible emojis.

## Markdown

Standard [GitHub Flavored Markdown](https://github.github.com/gfm/) is supported.

## MDX

Use React components in your content:

```mdx
<Info>MDX lets you use components inline.</Info>
```

See the [Components](/components) reference.

## Frontmatter

Add metadata at the top of each file. See [Frontmatter](/frontmatter).
## Implementation

- `api/src/bundler/mdx.ts` `parseMdx`
- `api/src/bundler/plugins/index.ts` — plugin registry
