Scaffolds a new docs.page project with docs.json, docs/index.mdx, and docs/next-steps.mdx in the target directory, prompting before overwriting existing files.

## Usage

```bash
docs.page init [path]
```

| Argument | Description |
|----------|-------------|
| `path` | Directory to initialize (defaults to `.`) |

## Files created

- `docs.json` — configuration file
- `docs/index.mdx` — home page
- `docs/next-steps.mdx` — getting started guide

Prompts before overwriting existing files.
## Surface

- **cli**: `docs.page init [path]`
## Implementation

- `packages/cli/src/commands/init.ts` — command
- `packages/cli/src/cli.ts` — registration
