The `@docs.page/cli` package provides commands for scaffolding documentation projects, validating configuration and content, and managing the documentation AI agent.

## Installation

Run commands with `npx` — no global install required:

```bash
npx @docs.page/cli <command>
```

## Commands

| Command | Description |
| --- | --- |
| [`docs init`](/cli/init) | Scaffold `docs.json` and starter MDX files |
| [`docs check`](/cli/check) | Validate config, links, and assets |
| [`docs agent`](/cli/agent) | Create or delete the documentation agent |

## Environment variables

| Variable | Description |
| --- | --- |
| `DOCS_PAGE_API_BASE` | Base URL for agent provisioning API calls (used by `docs agent create`) |

## Programmatic exports

The CLI package exports command registrars for embedding in other Node tools:

```typescript
import {
  registerInitCommand,
  registerCheckCommand,
  registerAgentCommand,
} from "@docs.page/cli";
```
