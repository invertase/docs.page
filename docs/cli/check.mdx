Validates a docs.page directory for a docs.json or docs.yaml file, route clashes among docs/**/*.mdx files, and broken relative links in MDX content.

## Usage

```bash
docs.page check [path]
```

## Checks performed

- `docs.json` or `docs.yaml` exists
- No duplicate routes among `docs/**/*.mdx` files
- Relative links in MDX resolve to existing pages

Exits with code `1` on errors.

<Info>

The same validation runs in [local preview](/local-previewing) via the exported `check()` function.

</Info>
