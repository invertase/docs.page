Supports configurable primary and background colors plus a defaultTheme preference, with client-side light/dark mode toggling in the documentation header.
## Configuration keys

- `theme.defaultTheme`
- `theme.primary`
- `theme.primaryLight`
- `theme.primaryDark`
- `theme.backgroundLight`
- `theme.backgroundDark`

## Example

```json
{
  "theme": {
    "defaultTheme": "light",
    "primary": "#ECC918",
    "primaryLight": "#BFA213"
  }
}
```

Visitors can toggle light/dark mode from the documentation header.
## Implementation

- `api/src/config/models/theme.ts` — config schema
- `website/src/components/Theme.tsx` — toggle UI
